# grit-taskmanager

Grit — task manager coding assignment.

## Run on localhost

From this folder, start a static server bound to **127.0.0.1** (so `localStorage` stays on one origin):

```bash
npm start
```

Then open **http://127.0.0.1:8080/** in your browser.

Alternatively:

```bash
sh serve.sh
```

Use **http://127.0.0.1:8080/** every time (not `file://`, and don’t mix `localhost` vs `127.0.0.1` if you care about persisted data).
